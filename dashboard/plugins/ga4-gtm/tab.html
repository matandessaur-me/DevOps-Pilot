GA4 & GTM Analytics

Connect your Google account to access Analytics and Tag Manager data.

Follow the steps below to set up Google Cloud OAuth credentials.

Step 1 -- Create a Google Cloud project
Go to Google Cloud Console > New Project and create a project (or use an existing one).
Step 2 -- Enable the required APIs
Enable each of these APIs for your project (click each link and press "Enable"):
  • Google Analytics Admin API
  • Google Analytics Data API
  • Tag Manager API
Step 3 -- Configure the OAuth consent screen
Go to OAuth consent screen and configure it:
  • User type: External
  • App name: anything (e.g. "DevOps Pilot")
  • Under Scopes, click "Add or Remove Scopes" and add these three:
    https://www.googleapis.com/auth/analytics.readonly
    https://www.googleapis.com/auth/tagmanager.edit.containers
    https://www.googleapis.com/auth/userinfo.email
  • Under Test users, add your Google email address
Step 4 -- Create OAuth credentials
Go to APIs & Services > Credentials, click Create Credentials > OAuth client ID:
  • Application type: Web application
  • Name: anything (e.g. "DevOps Pilot")
  • Authorized redirect URI -- add exactly:
    http://127.0.0.1:3800/api/plugins/ga4-gtm/auth/callback
Step 5 -- Add credentials to DevOps Pilot
Copy the Client ID and Client Secret from the credential you just created, then paste them into Settings > Plugins > GA4 & GTM Analytics.
